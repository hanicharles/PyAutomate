**📘 3. Unstructured vs. Structured Data**

📚 **What is Unstructured Data?**
Unstructured data is data that does not follow a specific format or schema, such as images, videos, and text documents.

✅ **Key Features:**
✅ No fixed schema
✅ No predefined structure

📊 **Example:**
An image file is an example of unstructured data.

📚 **What is Structured Data?**
Structured data is data that follows a specific format or schema, such as tables, spreadsheets, and databases.

✅ **Key Features:**
✅ Fixed schema
✅ Predefined structure

📊 **Example:**
A table with columns and rows is an example of structured data.

📚 **What is NoSQL?**

✅ JSON-like data storage

📊 **Example:**
MongoDB is an example of a NoSQL database.

📚 **What is SQL?**
SQL is a type of database that uses the traditional table-based relational model.

✅ **Key Features:**
✅ Fixed schema
✅ Predefined structure
✅ SQL query language

📊 **Example:**
A relational database management system (RDBMS) is an example of a SQL database.
